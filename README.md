# Bilal Shakeel Cinematic Portfolio

A premium, cinematic travel influencer portfolio built with React, Vite, Tailwind CSS (v4), and Framer Motion. 

Features an alternating storytelling layout, 3D floating media cards, immersive cinematic shadows, and responsive parallax video integration.

## Installation

```bash
npm install
npm run dev
```

## Built With
* React 19
* Vite 6 (TypeScript)
* Tailwind CSS
* Framer Motion
* Lucide React
